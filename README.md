SnapClass — AI Powered Attendance System

Overview

SnapClass is an advanced AI-based attendance platform that automates classroom attendance using Facial Recognition, Voice Biometrics, and QR-based Enrollment.

The system is designed for modern educational institutions to eliminate manual attendance processes while improving speed, accuracy, and security.
Built using:

Streamlit
Flask
Supabase
Face Recognition
Resemblyzer
Librosa
Dlib

The project includes:

AI Face Attendance
Sequential Voice Attendance
QR-based Enrollment
Attendance Analytics
Student & Teacher Dashboards
Secure Authentication

Based on the uploaded landing page implementation.

Features
AI Face Recognition Attendance
Detects multiple students from a single classroom image
Uses facial embeddings for identity matching
Fast and accurate attendance marking
Powered by Dlib + FaceRecognition
Voice Biometric Attendance
Students say “Present” sequentially
AI verifies identity using voice embeddings
Real-time voice matching system
Powered by Resemblyzer + Librosa
QR-Based Enrollment
Unique QR code generated for each subject
Students instantly enroll by scanning
No manual subject registration required
Teacher Dashboard

Teachers can:

Create and manage subjects
View attendance logs
Start AI attendance sessions
Download CSV reports
Track student attendance trends
Student Dashboard

Students can:

Join subjects
Register FaceID & VoiceID
Track attendance percentage
View attendance history
